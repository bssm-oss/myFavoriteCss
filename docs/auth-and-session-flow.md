# Provider configuration flow

## English

Morph UI no longer has a product auth/session backend flow.

Current flow:

1. user opens the side panel
2. user stores a provider API key locally
3. background worker uses that key directly for provider planning

## 한국어

Morph UI에는 더 이상 제품 auth/session 백엔드 플로우가 없습니다.

현재 흐름:

1. 사용자가 side panel을 엽니다
2. 사용자가 provider API key를 로컬에 저장합니다
3. background worker가 그 key를 직접 사용해 planning을 수행합니다
