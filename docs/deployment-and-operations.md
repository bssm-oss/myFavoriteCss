# Deployment and operations

## English

Morph UI deployment is now simpler:

- build and package the extension
- deploy the static web app if you want hosted help/privacy/fixture pages

There is no Morph UI backend service to deploy.

## 한국어

Morph UI 배포는 이제 더 단순합니다.

- extension을 빌드하고 패키징
- hosted help/privacy/fixture 페이지가 필요하면 정적 웹앱 배포

배포해야 할 Morph UI 백엔드 서비스는 없습니다.
