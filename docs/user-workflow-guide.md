# User workflow guide

## English

1. Open the side panel.
2. Add a provider API key locally.
3. Enable the current site.
4. Choose a profile.
5. Preview.
6. Apply or undo.
7. Revisit the page to confirm local cache reuse.

## 한국어

1. side panel을 엽니다.
2. provider API key를 로컬에 추가합니다.
3. 현재 사이트를 enable합니다.
4. 프로필을 고릅니다.
5. preview를 실행합니다.
6. apply 또는 undo를 사용합니다.
7. 페이지를 다시 열어 로컬 캐시 재사용을 확인합니다.
