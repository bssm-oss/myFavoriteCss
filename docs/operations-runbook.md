# Operations runbook

## English

### Provider outage

- keep local cache usable
- communicate that fresh planning is degraded
- advise switching provider or using strict-local mode

### Bad key configuration

- clear the local provider config
- re-enter the key
- verify the selected model name

## 한국어

### Provider 장애

- local cache 사용 가능 상태 유지
- fresh planning 저하 상태를 명확히 알림
- provider 변경 또는 strict-local mode 사용 안내

### 잘못된 key 설정

- 로컬 provider config 삭제
- key 재입력
- 선택된 model 이름 확인
