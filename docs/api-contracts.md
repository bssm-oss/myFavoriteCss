# Contracts

## English

The main contracts in extension-only mode are local typed contracts, not backend HTTP APIs.

Important schema groups:

- page summary and fingerprint
- preference profiles and site settings
- provider capabilities and local provider config
- transform plan and compiled transform
- runtime message payloads between extension surfaces

## 한국어

extension-only 모드의 핵심 계약은 백엔드 HTTP API가 아니라 로컬 typed contract입니다.

중요한 스키마 그룹:

- page summary와 fingerprint
- preference profile과 site setting
- provider capability와 local provider config
- transform plan과 compiled transform
- extension surface 사이의 runtime message payload
