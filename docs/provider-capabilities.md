# Provider capabilities

## English

Morph UI exposes provider capabilities exactly as implemented in extension-only mode.

### OpenAI

- direct API usage with user-supplied key: supported
- key/model validation before save: supported
- structured output: supported
- vision input: supported by capability model
- official OAuth consumer linking: not implemented
- ChatGPT subscription reuse: unsupported

### Gemini

- direct API usage with user-supplied key: supported
- key/model validation before save: supported
- structured output: supported
- vision input: supported by capability model
- official OAuth consumer linking: not implemented
- Gemini Advanced subscription reuse: unsupported

## 한국어

Morph UI는 extension-only 모드에서 실제 구현된 provider capability만 노출합니다.

### OpenAI

- 사용자 제공 key 기반 직접 API 사용: 지원
- 저장 전 key/model 검증: 지원
- structured output: 지원
- vision input: capability 관점에서 지원
- 공식 OAuth consumer linking: 미구현
- ChatGPT 구독 재사용: 미지원

### Gemini

- 사용자 제공 key 기반 직접 API 사용: 지원
- 저장 전 key/model 검증: 지원
- structured output: 지원
- vision input: capability 관점에서 지원
- 공식 OAuth consumer linking: 미구현
- Gemini Advanced 구독 재사용: 미지원
