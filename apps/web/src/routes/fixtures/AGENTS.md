# Fixture route agent instructions

## English

These instructions apply to `apps/web/src/routes/fixtures`.

- fixtures are test infrastructure, not marketing pages
- preserve realistic but stable DOM structure
- keep landmark usage, headings, forms, and repeated blocks intentional
- when changing a fixture, consider E2E, smoke evidence, and manual verification docs together

## 한국어

이 지침은 `apps/web/src/routes/fixtures`에 적용됩니다.

- fixture는 테스트 인프라이지 마케팅 페이지가 아닙니다
- 현실적이면서도 안정적인 DOM 구조를 유지합니다
- landmark 사용, heading, form, repeated block은 의도적으로 유지합니다
- fixture를 바꾸면 E2E, smoke evidence, manual verification 문서를 같이 검토합니다
