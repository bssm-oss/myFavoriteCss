# Shared contract agent instructions

## English

These instructions apply to `packages/shared`.

- this is the contract authority for Morph UI
- schema changes are product changes
- keep types, schemas, and exported contracts aligned
- prefer explicit defaults over implicit optional drift
- if a contract changes, review docs and all callers

## 한국어

이 지침은 `packages/shared`에 적용됩니다.

- 이 패키지는 Morph UI 계약의 정본입니다
- schema 변경은 곧 제품 변경입니다
- 타입, schema, export contract를 항상 맞춰 유지합니다
- 암묵적 optional drift보다 명시적 default를 선호합니다
- contract가 바뀌면 문서와 모든 caller를 함께 검토합니다
